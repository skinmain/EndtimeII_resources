## 关于整合包
该整合包最早立项于2023年10月<br>
因为作者厌恶了制作传统丧尸整合包（但此类整合包我只有踪灭一有发布），在b站看到了相关模组的视频后，便开始了制作。<br>
起初只是想为自己换一个和惊变丧尸强度差不多的对手，但在我接触了Changed和一些与福瑞相关的东西后......<br>
因为一些比较复杂的经历，最终奠定了该整合包现在的创作基调。<br>
一切改动和模组的选择均为整合包的故事背景以及整个系列的独立世界观服务，并且尽可能最大程度地优化游戏性能<br>
有关胶兽的机制修改（已确认不包含版权纠纷）部分灵感来源于Changed原作、现实动物研究数据和俺寻思之力。<br>
适合喜欢挑战极限以及寂静末日风格的玩家（虽然氛围还差点意思）。<br>
