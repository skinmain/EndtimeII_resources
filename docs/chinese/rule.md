# 内容发布准则
当你开始加入社区并进行问题报告和内容发布等发布行为时，即视为同意本规则的全部条款。如有违反，作者将视情况采取相应处理（包括但不限于拉黑、删除内容等）.<br><br>
若其他规定与本规则存在模糊或冲突之处，以本规则为准。如果整合包内所包含的模组的作者另有要求，则以该模组作者的要求为准<br> <br>
本规则（除下面的补充规则）属于整合包投稿的**基本底线**，请务必遵守。对于严重违规行为，作者将保留进一步采取措施的权利。<br>

## 准则
1. **严禁添加违法违规或严重违背公序良俗的内容**  
   包括但不限于：政治敏感(包括**现实政治立场、事件、宣传、符号等**)、暴力、黄赌毒、造谣抹黑、乱伦、反动、歧视等违反国际法律法规及公序良俗、违反国家法律、或明显反人类、反社会、反道德的内容。<br>
   *此类内容在任何渠道都可能造成不良影响，请绝对不要投稿。*

2. **严禁发布社区敏感内容**  
   指可能引发该整合包社区或其他社区（如 Changed 原作社区）争议、对立或不良影响的内容，例如强行关联 Changed 原作的世界观等。<br>
   *请尊重不同社区的边界，避免制造无谓的冲突。*

3. **请注意发布场合**  
   请勿在与整合包无关的场合（如完全不相关的讨论群、视频评论区、其他作者的整合包社区等）反复或强行推广本整合包及其内容。<br>
   *我们不希望因为过度宣传而给他人带来困扰。*

4. **严禁针对整合包相关人员的不当行为**  
   包括但不限于：散布关于整合包、作者或任何贡献者的不实谣言、泄露他人隐私、主动引战或进行人身攻击等。<br>
   *请保持基本的尊重和理性沟通，如有争议可通过正规渠道（如 Issue、官方交流群、作者B站等整合包官方交流渠道）解决。*

5. **其他可能对整合包及其社区造成严重不良影响的行为**  
例如：恶意利用 bug 或编写恶意文件破坏他人游戏体验、组织针对作者或社区的骚扰活动等。  
  *此类行为虽难以完全列举，但明显违背社区安全与友善原则。*

## 其他补充

以下内容不属于严格的“红线”，但遵守它们能让社区氛围更好，也减少不必要的冲突。作者会优先进行提醒，多次或恶意违反者可能被限制交互（如拉黑）。

- 报告 bug 或问题时，请尽量包含以下内容：整合包版本、Minecraft 版本、发生场景、是否安装其他额外模组、有无错误日志等。<br>
    *“玩不了”、“卡死了” 这类信息对解决问题几乎没有帮助。*<br>

- 理性交流，禁止辱骂、人肉、骚扰、嘲讽作者或其他玩家。<br>
    *作者和其他贡献者也都是普通玩家，用爱发电，互相尊重是基本底线。*<br>

- 请勿私下拆包、二次发布本整合包或未授权的修改版。<br>
    *如需参考或使用部分内容，请联系作者取得许可。*<br>

- 保护个人隐私，不要在公开渠道泄露敏感信息，也不要向他人索取相关信息。<br>
    *社区只关心整合包本身，不需要这些信息。*<br>

- 其他可能对各方造成不利影响的行为  
*为他人，也为自己的利益*
