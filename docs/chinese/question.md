# 相关问题

1. **有关Changed?** <br>
本整合包使用了由 **Ltxprogrammer** 开发的 **Changed-Minecraft-Mod（MIT 许可证）**模组。  
该模组**仅作为本包玩法框架的一个组成部分**，其提供的**生物、物品等素材**均在本包独立构建的末日世界观下**被重新赋予背景与功能**。  
***本整合包的整体剧情、任务线、建筑风格及核心设定，均为作者完全独立的虚构创作，与《Changed》原作剧情及任何同人设定无任何关联。***  
<br>

作者的个人生活与创作动机无关，**谢绝就此类私人话题进行询问或讨论**，请尊重创作者的基本隐私边界。  
<br>

3. **政治内容?**<br>

本包所集成的第三方模组，作者已对其界面文本与显示内容进行筛查，无任何政治敏感内容。

4. **有关东欧风格？**<br>
本整合包的视觉风格选择，**纯粹基于末日生存题材下的审美偏好**——粗野主义混凝土废墟的压迫感、工业设施的荒凉感，以及光影下的独特氛围，符合作者对本作“冷寂末日”的构想。  
这种**建筑与相关美学风格**的参考，不涉及任何现实地区、政权或历史时期的社会制度或意识形态。**作者拒绝将艺术风格与政治立场进行任何形式的绑定与解读。**  
<br>

5. **游戏设置？**<br>
考虑到玩家不同的游玩习惯及作者自身的游戏设置非常逆天，本整合包**不自带预设设置**，请各位游玩之前先调整下 *画面设置* 及 *键位设置*。
<br>

6. **光影包？**<br>
鉴于许可限制及不同玩家考虑，本整合包自beta1.1起不再在导入包中自带光影包，如有需要，请各位自行下载适合自己的光影。

7. **配置问题？**<br>
你可以通过修改整合包内的配置文件及部分内容来优化自己的游戏体验，毕竟玩的开心才是最重要的。<br>
注意：不可二次分发该整合包。<br>
