# 相关问题

<br>

1. **有关Changed?** <br>
该整合包使用了由<a href="https://github.com/LtxProgrammer">Ltxprogrammer</a>制作的<a href="https://github.com/LtxProgrammer/Changed-Minecraft-Mod">Changed-minecraft mod</a>模组(其使用MIT许可),所有相关制作均基于该模组制作。<br>
任何可能与Changed原作相关联的内容均做出了修改，仅包含模组提供内容及部分机制。<br>
整合包的所有世界观和故事/背景均为独立开发，纯属虚构，由作者自行创作，和Changed原作没有任何关联。<br>
请不要在各个平台的相关内容下提与该整合包相关的无关内容，还恳请各位注意言行，谢谢！<br>
<br>

2. **有关福瑞?**<br>
由于作者有相关的被霸凌经历以及童年经历，<br>
请避免询问作者与福瑞相关的个人经历，谢谢理解。<br>
<br>

3. **政治内容?**<br>
整合包内的所有内容和故事内容均经过作者严格审查，整合包内的魔改内容均没有和现实有关的政治内容<br>
作者会定期检查内容并清除，其他模组添加的政治内容作者会及时删除<br>
如果你游玩时发现了政治内容，请及时联系作者处理。<br>
<br>

4. **有关东欧风格？**<br>
由于作者看腻了美式风格，所以作者便参考了部分与东欧相关的内容进行**虚构创作**，**绝非政治原因**。<br>
如果你发现有人利用本整合包相关内容进行违反社区规定的行为，欢迎通过 issue 或联系作者反馈，感谢你的帮助。<br>

5. **游戏设置？**<br>
考虑到玩家不同的游玩习惯及作者自身的游戏设置非常逆天，本整合包不自带预设设置，请各位游玩之前先调整下 *画面设置* 及 *键位设置*。
<br>

6. **光影包？**<br>
鉴于许可限制及不同玩家考虑，本整合包自beta1.1起不再在导入包中自带光影包，如有需要，请各位自行下载适合自己的光影。

7. **配置问题？**<br>
你可以通过修改整合包内的配置来优化自己的游戏体验，毕竟玩的开心才是最重要的。<br>
