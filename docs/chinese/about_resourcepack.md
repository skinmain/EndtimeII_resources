# 资源包
全局资源包由于OpenLoader模组的机制，无法通过原版方法禁用。  
这些资源包可以在 **[整合包游戏目录]/config/openloader/resources** 中找到。  

# 资源包文件名及其介绍

- **changed_pbr**  
Changed mod自带的资源包，用于支持光影的pbr功能。
- **VanillaTweaks_r920812_MC1.20.x.zip**  
作者用<a href="https://vanillatweaks.net/">VanillaTweaks</a>定制的资源包，游戏中的部分原版材质修改来源于此。  
不喜欢黑暗风格和圆形准星之类的材质修改的话，删除该资源包即可。
